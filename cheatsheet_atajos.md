# ⌨️ Guía Completa de Atajos de Teclado (Neovim)

Esta es la documentación actualizada de tu entorno de desarrollo Flutter en Neovim.

**Nota:** La tecla `<Lider>` (Leader key) es la tecla **Espacio**.

---

## 📂 Archivos y Navegación

| Atajo | Acción |
| :--- | :--- |
| `<Espacio> + e` | **Explorador de Archivos** (Abrir/Cerrar panel lateral) |
| `<Espacio> + ff` | **Buscar Archivos** (Por nombre) |
| `<Espacio> + fg` | **Buscar Texto** (En todo el proyecto - Grep) |
| `<Espacio> + fb` | **Buscar Buffers** (Pestañas abiertas) |
| `<Ctrl> + h/j/k/l` | Moverse entre ventanas (Izquierda/Abajo/Arriba/Derecha) |
| `<Ctrl> + s` | **Guardar** archivo |

## 🚀 Flutter y Dart

| Atajo | Acción |
| :--- | :--- |
| `<Espacio> + fn` | **Nuevo Proyecto** (Crear desde cero) |
| `<Espacio> + fe` | **Emuladores** (Lanzar Android/iOS) |
| `<Espacio> + fr` | **Run** (Ejecutar app - `flutter run`) |
| `<Espacio> + fR` | **Hot Restart** (Reiniciar app rápido) |
| `<Espacio> + fq` | **Quit** (Cerrar app en ejecución) |

## 🧠 Inteligencia de Código (LSP)

*Pon el cursor sobre una clase, función o variable:*

| Atajo | Acción |
| :--- | :--- |
| `gd` | **Ir a Definición** (Go to Definition) |
| `gD` | Ir a Declaración |
| `gr` | **Ver Referencias** (Dónde se usa esto) |
| `K` | **Ver Documentación** (Hover) |
| `<Ctrl> + k` | Ayuda de firma (parámetros de función) |
| `<Espacio> + rn` | **Renombrar** variable/función automáticamente |
| `<Espacio> + ca` | **Acción de Código** (Quick Fix / Refactor) |
| `<Espacio> + d` | Ver diagnósticos (errores) en ventana flotante |
| `[d` / `]d` | Saltar al error anterior / siguiente |

## 💻 Terminal y Git

| Atajo | Acción |
| :--- | :--- |
| `<Ctrl> + \` | **Terminal Flotante** (Abrir/Cerrar) |
| `<Espacio> + gg` | **LazyGit** (Interfaz visual completa para Git) |

## 📝 Edición General

| Atajo | Acción |
| :--- | :--- |
| `jk` | **Salir de modo Insertar** (Equivalente a Esc) |
| `<Espacio> + nh` | **No Highlight** (Limpiar resaltado de búsqueda) |
| `<Tab>` / `<S-Tab>` | Navegar en el menú de autocompletado |
